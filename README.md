# CornerView
可以通过xml配置以下属性改变圆角的位置
 - `cornerTopLeft`
 - `cornerTopRight`
 - `cornerBottomLeft`
 - `cornerBottomRight`
 
 或者通过调用以下方法来改变圆角的位置
 - `setCornerTopLeft`
 - `setCornerTopRight`
 - `setCornerBottomLeft`
 - `setCornerBottomRight`
